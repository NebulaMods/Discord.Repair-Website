First create a bot on the Discord developer portal. Then follow the steps how to create bot, then create a server & send a verify a message for users.
Discord Repair Bot Setup Once you have created your discord application press the OAuth2 button, copy the applications client secret, and paste it in the client secret textbox under create bot at https://discord.repair/dashboard/bots.html. After you have done the client secret go back to the OAuth2 page for your discord application and copy the applications client id, and paste it in the client id textbox under create bot at https://discord.repair/dashboard/bots.html. Now that you have got your client id done go back to the OAuth2 page in your discord application and add a redirect that points to "https://discord.repair/verify". Now that the OAuth2 stuff is out of the way go to the bot section in your discord application, add a bot if you have not already, and copy the discord bot token and paste it into the textbox named token under create bot at https://discord.repair/dashboard/bots.html. Finally, give your bot a name and if you did everything correctly it should look something like the image down below and you can press the create button.
